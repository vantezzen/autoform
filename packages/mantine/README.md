# Mantine integration for AutoForm

This package provides a Mantine integration for AutoForm.

## Installation

```bash
npm install @dual-autoform/mantine
```

Choose the form engine explicitly:

```tsx
import { AutoForm } from "@dual-autoform/mantine/react-hook-form";
// or: @dual-autoform/mantine/tanstack-form
```

For more information on how to use this package, see the [AutoForm documentation](https://autoform-dual.vercel.app/docs/react/getting-started).
