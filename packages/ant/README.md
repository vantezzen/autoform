# ANT integration for AutoForm

This package provides an Ant Design integration for AutoForm.

## Installation

```bash
npm install @acp-autoform/ant
```

Choose the form engine explicitly:

```tsx
import { AutoForm } from "@acp-autoform/ant/react-hook-form";
// or: @acp-autoform/ant/tanstack-form
```

For more information on how to use this package, see the [AutoForm documentation](https://autoform-acp-docs.vercel.app/docs/react/getting-started).
