# MUI integration for AutoForm

This package provides a MUI integration for AutoForm.

## Installation

```bash
npm install @dual-autoform/mui
```

Choose the form engine explicitly:

```tsx
import { AutoForm } from "@dual-autoform/mui/react-hook-form";
// or: @dual-autoform/mui/tanstack-form
```

For more information on how to use this package, see the [AutoForm documentation](https://autoform-dual.vercel.app/docs/react/getting-started).
