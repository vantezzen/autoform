# Chakra UI integration for AutoForm

This package provides a Chakra UI integration for AutoForm.

## Installation

```bash
npm install @acp-autoform/chakra
```

Choose the form engine explicitly:

```tsx
import { AutoForm } from "@acp-autoform/chakra/react-hook-form";
// or: @acp-autoform/chakra/tanstack-form
```

For more information on how to use this package, see the [AutoForm documentation](https://autoform-acp-docs.vercel.app/docs/react/getting-started).
