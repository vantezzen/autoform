## How do I submit, reset AutoForm from buttons in a Dialog?

Use either approach:

1.  Autoform wraps the form with [`FormProvider`](https://react-hook-form.com/docs/formprovider/). Render the dialog components as `AutoForm` children and use the [`useFormContext`](https://react-hook-form.com/docs/useformcontext/) hook to access form methods and states.

2.  Access form methods and states externally with [`createFormControl`](https://react-hook-form.com/docs/createFormControl) and trigger actions like submission, reset etc.

## How do I show validation errors in real time and disable submit until valid?

Create a formControl with `createFormControl` and set the validation mode to `onChange`. Subscribe to the form state to track validation status and disable the submit button accordingly.

## How do I create a simple multistep form?

Use one small schema for each step. Render the current step, save its values in state, and move forward from `onSubmit`. Since the Next button submits the current step, AutoForm validates that step before moving ahead.

## How do I use another AutoForm inside an AutoForm field?

Define a custom field component, register it in `formComponents`, then mark the schema field with `fieldConfig({ fieldType: "..." })`. When the nested AutoForm submits, pass its value back to the parent via `onChange`.

## How do I create a live interactive form builder?
