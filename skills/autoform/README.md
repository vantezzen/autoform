# @dual-autoform AI Agent Skill

This directory contains the official skill for **AutoForm** (`@dual-autoform`).

## Install

Install this skill:

```bash
npx skills add https://github.com/adityacodepublic/autoform/tree/tanstack-form-integration/skills --skill autoform
```

## Directory Structure

- **[`SKILL.md`](./SKILL.md)**: The primary entry point. It contains the essential architecture overview, installation commands, quick start examples, and critical anti-patterns ("Common Mistakes to Avoid").
- **[`references/`](./references/)**: Deep-dive guides.
  - Full installation matrix (`installation.md`)
  - Schema provider setup (`schema-providers.md`)
  - Using the Shadcn UI registry integration (`shadcn.md`)
  - Shadcn/ui and Tailwind installation from scratch (`shadcn-tailwind-installation.md`)
  - Advanced form state management (`form-control.md`)
  - Customizing fields and UI components (`customization.md`)
  - Building a custom UI integration (`custom-integration.md`)
