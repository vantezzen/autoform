# @autoform AI Agent Skill

This directory contains the official skill for **AutoForm** (`@autoform`).

## Install

Install this skill:

```bash
npx skills add https://github.com/vantezzen/autoform/tree/main/skills --skill autoform
```

## Directory Structure

- **[`SKILL.md`](./SKILL.md)**: The primary entry point. It contains the essential architecture overview, installation commands, quick start examples, and critical anti-patterns ("Common Mistakes to Avoid").
- **[`references/`](./references/)**: Deep-dive guides.
  - Full installation matrix (`installation.md`)
  - Schema provider setup (`schema.md`)
  - Shadcn/ui and Tailwind installation from scratch (`utils/shadcn-tailwind-installation.md`)
  - React Hook Form shadcn examples (`utils/examples-react-hook-form.md`)
  - TanStack Form shadcn examples (`utils/examples-tanstack-form.md`)
  - React Hook Form adapter patterns (`react-hook-form.md`)
  - TanStack Form adapter patterns (`tanstack-form.md`)
  - Customizing fields and UI components (`customization.md`)
  - Building a custom UI integration (`utils/custom-ui-package-integration.md`)
  - Copying the React adapter source (`utils/customizing-react-package.md`)
